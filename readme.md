# Telegram TaskBot
Телеграм бот на **Python** и **aiogram**,
является твоим помошником в некоторых мелких задачах,
которые могут тратить твой фокус внимания.

### Features(soon)
- Напоминания
- Погода(исользует ваше местоположение | название города)
- Предупреждение о дожде, высоком уф-индексе, ...
- Собственный трекер книг('прочитано', 'в планах', 'читаю')

### Tech Stack
**Stack:** python 3.14.5, asyncio, asyncpg,
    alembic, sqlalchemy2, pydantic, aiogram

### Getting Started
**Установка и запуск:**
```Bash
# 1. Клонируйте репозиторий
git clone https://github.com/vladislav-png/название-проекта.git

# 2. Перейдите в директорию проекта
cd .../tgBot/

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Создайте файл настроек из шаблона
cp .env.example .env

# 5. Запустите main.py
python3 src/main.py
```
